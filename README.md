## Description
This program implements dynamic task distribution using multiprocessing in Python.

## Method
- Tasks are stored in a queue
- Workers take tasks dynamically during runtime
- Load balancing occurs automatically

## Result
- Each worker processes different number of tasks
- Execution time is shown
- Optimal time achieved because no worker is idle

## How to Run
python dynamic.py
